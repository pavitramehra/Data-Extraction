# SQL_to_CSV
A GUI to convert SQL db in to CSV files.

1. To install all dependencies in local system, run 
```
  pip install -r requirments.txt
 ```
2. To start GUI in local, run 
```
  python app.py
```
**Note:** make sure you are in right directory.
